## Selezione di righe

Per selezionare righe in base alla loro posizione, Pandas offre i metodi `.loc[]` e `.iloc[]`. Mentre `.loc[]` si riferisce agli indici basati su etichetta, `.iloc[]` si riferisce agli indici basati su posizione (interi):

```python
# Seleziona la riga con etichetta indice 5
riga = df.loc[5]

# Seleziona la sesta riga (base 0)
riga = df.iloc[5]
```

## Filtraggio condizionale

Spesso, si desidera selezionare righe in base a una o piÃ¹ condizioni. Ecco come fare:

```python
# Supponiamo di voler selezionare tutte le righe dove la colonna 'eta' Ã¨ maggiore di 18
maggiorenni = df[df["eta"] > 18]
```

Per combinare piÃ¹ condizioni, si possono utilizzare gli operatori `&` (and) e `|` (or):

```python
# Seleziona le righe dove 'eta' Ã¨ maggiore di 18 e 'cittÃ ' Ã¨ "Roma"
roma_maggiorenni = df[(df["eta"] > 18) & (df["cittÃ "] == "Roma")]
```

Ricorda di mettere ogni condizione tra parentesi quando combini le condizioni.
