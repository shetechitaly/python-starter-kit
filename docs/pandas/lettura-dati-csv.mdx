## read_csv()

La funzione read_csv() di Pandas √® una delle funzioni pi√π utilizzate per caricare dati da file CSV. Ha molte opzioni che possono essere personalizzate per gestire casi particolari, ma l'uso base √® abbastanza semplice.

Ecco un esempio di come leggere un file CSV:

```python
import pandas as pd

# Leggi il file CSV e memorizzalo in un DataFrame
df = pd.read_csv('nome_del_file.csv')

# Stampa le prime 5 righe del DataFrame
print(df.head())
```

Nell'esempio sopra, stiamo leggendo un file CSV chiamato 'nome_del_file.csv' e stampando le prime 5 righe del DataFrame risultante.

## Opzioni Comuni

La funzione read_csv() ha molte opzioni che possono essere utili. Ecco alcune delle pi√π comuni:

**delimiter**: Specifica il delimitatore utilizzato nel file. Il default √® la virgola (,). Ad esempio, per un file TSV (Tab-Separated Values), puoi usare delimiter='\t'.

**header**: Specifica quale riga del file da utilizzare come intestazione delle colonne. Di default √® impostato su header=0, che utilizza la prima riga.

**index_col**: Permette di specificare una colonna del CSV da utilizzare come indice per il DataFrame.

**usecols**: Consente di selezionare un sottoinsieme di colonne da leggere nel DataFrame.

Ecco come potresti utilizzare alcune di queste opzioni:

```python
df = pd.read_csv('nome_del_file.csv', delimiter=';', header=None, usecols=[0, 2, 4])
```
