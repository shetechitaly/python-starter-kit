# Statistiche descrittive semplici
Argomenti: media, mediana, deviazione standard, quartili, valori minimi e massimi.

## Intro

Una parte essenziale dell'EDA Ã¨ comprendere le statistiche descrittive di base dei dati. Queste statistiche forniscono un riassunto numerico delle caratteristiche di un set di dati, permettendo di ottenere rapidamente una visione d'insieme delle tendenze centrali, della dispersione e della forma dei dati.

## Statistiche chiave

### Media e Mediana

**Media** Ã¨ la somma di tutti i valori divisa per il numero di valori. Riflette il valore "centrale" del set di dati.

```python
media = data['colonna'].mean()
```

**Mediana** Ã¨ il valore centrale di un set di dati quando Ã¨ ordinato. Ãˆ meno sensibile agli outlier rispetto alla media.


```python
mediana = data['colonna'].median()
```

### Deviazione Standard

La **deviazione standard** rappresenta la dispersione dei dati attorno alla loro media. Un valore elevato indica una maggiore variabilitÃ .

```python
dev_std = data['colonna'].std()
```

### Quartili, Minimi e Massimi

I **quartili** dividono i dati in quattro parti uguali. Il primo quartile (Q1) Ã¨ il valore al 25Â° percentile, mentre il terzo quartile (Q3) Ã¨ al 75Â° percentile.

```python
Q1 = data['colonna'].quantile(0.25)
Q3 = data['colonna'].quantile(0.75)
```

I **valori minimi e massimi** rappresentano i valori estremi del set di dati.

```python
minimo = data['colonna'].min()
massimo = data['colonna'].max()
```

:::tip 
Il metodo `describe()` di Pandas fornisce un sommario di tutte queste statistiche descrittive in una sola chiamata:

```python
data['colonna'].describe()
```
:::
